## 🚀 Getting Started

### 📋 Prerequisites

- **Git**: Version 2.20+ ([Download here](https://git-scm.com/downloads))
- **GitHub Account**: [Sign up for free](https://github.com/join)
- **Terminal/Command Line**: Basic familiarity recommended

### 🏃‍♂️ Quick Start

```
# Clone the repository
git clone https://github.com/AshC2004/How_to_Github.git

# Navigate to the directory
cd How_to_Github

# Start with the basics
cat Setup.md
```

### 🚀 Contribution Process

1. **🍴 Fork** the repository
2. **🌱 Create** a feature branch (`git checkout -b feature/amazing-improvement`)
3. **💻 Make** your changes
4. **✅ Test** your changes
5. **📝 Commit** with a descriptive message (`git commit -m 'Add amazing improvement'`)
6. **🚀 Push** to your fork (`git push origin feature/amazing-improvement`)
7. **🔀 Create** a Pull Request

### 📋 Contribution Guidelines

- Include examples for new concepts
- Test all code snippets
- Update the table of contents if needed
